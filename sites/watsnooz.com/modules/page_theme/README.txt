Module
======
Page Theme


Description
===========
This module allows to use different themes than the site default on specific
pages.


Requirements
============
Drupal 6.x


Installation
============
1. Move this folder into your modules directory.
2. Enable it from Administer >> Site building >> Modules >> Theme related.


Configuration
=============
Configure it at Administer >> Site building >> Page theme.


Projekt page
============
http://drupal.org/project/page_theme
